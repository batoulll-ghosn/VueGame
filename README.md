## Getting Started
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
source ~/.nvm/nvm.sh

nvm install 10.24.1                                                            
nvm use 10.24.1     

npm install 

npm run build 

npm run serve
